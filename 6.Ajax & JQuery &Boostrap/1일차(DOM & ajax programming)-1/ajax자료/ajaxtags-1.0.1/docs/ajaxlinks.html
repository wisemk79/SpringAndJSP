Ajax Resources and Links on the Web

Ajax is not necessarily new, but the recent explosion in development using Ajax may lead you to think otherwise. Below is a list of resources we've found to be very informative on the topic of Ajax and should help you get up to speed quickly. While we don't intend to keep a complete directory of resources--others do that better--feel free to drop us a line if you have suggestions for improving this list. Note that some of this is Java-centric.

Notice: there is another project currently in the alpha stages of development that uses the AjaxTags moniker in one of its submodules. We've included a link to that project below. As a matter of clarification, the AJAX JSP Tag Library (aka, AjaxTags) is in no way affiliated with the AjaxTags module of the Java Web Parts project.

  • Ajax Patterns - effort to document best practices for AJAX
  • Ajaxian - keep up to date with the latest in AJAX with this blog from Dion and Ben (or is it Ben and Dion)
